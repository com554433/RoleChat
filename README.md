# RoleChat

一款桌面角色扮演聊天软件——输入角色名和作品名，即可自动创建角色并对话。

## 能做什么

**一键创建角色** · **扮演角色语音合成** · **多角色切换** · **深色/浅色主题**

---

## 快速开始

双击 `RoleChat.exe` 打开，点击左侧 `+` 号导入角色。

如果没有角色，可以在导入面板输入：
- **角色名**：比如"孙悟空"
- **作品名**：比如"西游记"

点击生成，等待几秒即可开始聊天。

## 设置

点击右上角齿轮图标，需要填写：

| 项目 | 说明 |
|------|------|
| 服务密钥 | 服务商提供的账号凭证 |
| 服务地址 | 服务商提供的访问地址 |
| 版本号 | 默认 `mimo-v2.5-pro` |

还有一个备用账号开关，用于切换不同配置，一般不用管。

语音功能需要上传一段角色声音样本，开启后角色会用该声音朗读回复。
