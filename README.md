# RoleChat

一款桌面角色扮演聊天软件——输入角色名和作品名，即可自动创建角色并对话。

## 能做什么

**一键创建角色** · **角色语音朗读** · **多角色切换** · **深色/浅色主题**

---

## 快速开始

双击 `RoleChat.exe` 打开，点击左侧 `+` 号导入角色。

如果没有角色，可以在导入面板输入：
- **角色名**：比如"孙悟空"
- **作品名**：比如"西游记"

点击生成，等待几秒即可开始聊天。

## 设置

点击右上角齿轮图标，填写服务凭证和地址即可使用。

语音功能需要上传一段角色录音，开启后角色会用该声音朗读回复。
